# Belajar-Git-Github
Mendokumentasikan Pembelajaran Menggunakan tools git dan github, kayaknya ini ga perlu dibuat file 

Saya hanya belajar fungsi / istilah dasar, berikut yang saya pelajari:

1. Konfigurasi Awal
git config --global user.name "Nama Kamu"
git config --global user.email "email@example.com"
git config --list

2. Membuar repository
git init

3. mengecek status file
git status

4. Menambahkan ke stagging area
git add namafile
git add .

5. menyimpan perubahan
git commit -m "pesan apa"

6. melihat riwayat commit
git log
git log --oneline

7. menghubungkan repository ke github

git push
git remote

8. Menyalin repository
git clone
fork
